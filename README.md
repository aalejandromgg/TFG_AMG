# TFG_AMG
Repositorio para el TFG de Alejandro Muñoz García: "Detección de anomalías mediante distribuciones alfa-estables y técnicas de aprendizaje automático"

- Obtención del conjunto de datos: El conjunto de datos que hemos utilizado ha sido el UGR'16 (https://nesg.ugr.es/nesg-ugr16/), el cual nos proporciona unas series temporales para diferentes semanas.
